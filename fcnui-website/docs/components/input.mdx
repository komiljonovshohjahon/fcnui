# Input

Displays a form input field or a component that looks like an input field.


## Installation
```shell
fcnui add input
```

> :warning: **Important**:
> When added, need to stop running app and run `flutter pub get` to install all necessary dependencies.


## Depends

Adds the following components to your project along with the input component:

- [Form](/docs/components/form)
- WithLabel
- [SaveButton](/docs/components/form#additionally)
